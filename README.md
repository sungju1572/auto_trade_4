# auto_trade_4

______________________
# 버전체크
아나콘다 32bit `4.10.1`

# 필요 라이브러리
pip install pykiwoom

2. Kiwoom_2.py (키움증권 연결부분 코드)
3. pytrader.ui (ui 디자인)
