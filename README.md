# auto_trade_4
영웅문 조건검색식에 따른 실시간 종목 거래


______________________
# 버전체크
아나콘다 32bit `4.10.1`

# 필요 라이브러리
pip install pykiwoom
____________________


# 실시간 거래 사용 파일
1. pytrader.py (ui부분 코드)
2. Kiwoom.py (키움증권 연결부분 코드)
3. exam_2.ui (ui 디자인)
